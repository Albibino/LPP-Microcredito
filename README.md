Trabalho de linguagens de programação e paradigmas - Trabalho 2

Tema: Microcrédito (Juros compostos, adimplência e atrasos)

Sobre o projeto:
- Simulador simples do tema de microcrédito, teste parcelas, valores, juros, pagamentos, e politicas de pagamentos.
- Programação funcional

Como usar:
- Python 3.9+
- git clone "https://github.com/Albibino/LPP-Microcredito"
- python -m venv venv
- .venv\Scripts\activate
- pip install -r .\requirements.txt
- python app/main.py
